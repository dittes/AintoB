About the Formats

🎞️

GIF (Graphics Interchange Format)

GIF is one of the oldest web image formats, dating back to 1987. It supports simple frame-by-frame animations and is limited to a 256-colour palette per frame.

Compatibility: Universal — supported in every browser, email client, and messaging platform.

🖼️

AVIF (AV1 Image File Format)

AVIF is a next-generation image format developed by the Alliance for Open Media. It offers significantly better compression than JPG or WebP while maintaining high visual quality, supporting HDR, wide color gamut, and transparency.

Compatibility: Chrome 85+, Firefox 93+, Safari 16+, Edge 121+.

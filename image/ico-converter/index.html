What is ICO?

ICO is the icon container format used by Windows and by every website favicon. A single ICO file can hold multiple image sizes (16×16 up to 256×256) so the operating system or browser can pick the sharpest variant for the context.

Common uses for ICO

  • Website favicons served as /favicon.ico
  • Windows application and shortcut icons
  • Toolbar and system-tray icons at multiple sizes
Note: ICO is only meant for small icons — images are capped at 256×256 pixels, so it is unsuitable as a general-purpose image format.

Compatibility: Universal favicon support in every browser. Windows native. Most image editors need a plugin to save ICO.

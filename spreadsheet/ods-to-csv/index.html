About the Formats

📉

ODS (OpenDocument Spreadsheet)

ODS is the open-standard spreadsheet format used by LibreOffice, OpenOffice, and Google Sheets. Part of the ISO-standardised OpenDocument Format, ODS stores spreadsheet data — cells, formulas, charts, and formatting — in an open, non-proprietary way, free from vendor lock-in.

Compatibility: Native in LibreOffice and OpenOffice. Supported by Google Sheets, Microsoft Excel 2007+, and most modern spreadsheet tools.

📊

CSV (Comma-Separated Values)

CSV is a plain-text format for tabular data where each line represents a row and commas separate column values. It is the most universal data-exchange format — virtually every database, spreadsheet application, CRM, and analytics tool can import and export CSV files without any special configuration.

Compatibility: Universal — supported by Excel, Google Sheets, LibreOffice, every database system, and virtually all data tools.

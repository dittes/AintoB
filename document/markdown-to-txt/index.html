About the Formats

📝

Markdown (Markdown)

Markdown is a lightweight, human-readable plain-text formatting syntax created by John Gruber and Aaron Swartz. Simple punctuation marks up headings, bold, italics, links, code, and lists — and it converts cleanly to HTML.

Compatibility: Supported by GitHub, VS Code, Notion, Obsidian, Hugo, Jekyll, and virtually all developer tools.

📃

TXT (Plain Text)

Plain text (.txt) files contain unformatted characters with no styling, fonts, or layout.

Compatibility: Universal — readable by every OS, text editor, browser, and programming language.
